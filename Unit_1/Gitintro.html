Command Description 
git init
 Initializes our repo(repository) on our machines 
 git status
 Displays Tracked Files and their status 
git add
git add.
Stages any CHanges Files, add to the commit 
 git push
 Uploads your local commits 
 git pull
 Downloads or "pull down" any changes from GitHut 
 gir remote add origin https://github.com/userName/nameOfGitHubRepo
Connects URL origin to local project 
 git branch -M main
 Renames the master branch to main 
 git push -u origin main
 For initial push, sends code up repo for the first time.

    When Updating a Repo

  • git add .
  • git commit -m " message here"
  • git push origin main
